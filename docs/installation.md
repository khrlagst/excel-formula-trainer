# 📦 Installation

### 📦 Installation

To install `docgen` and its dependencies, ensure you have Python 3.11 or higher installed.

#### Prerequisites

- Python >= 3.11
- Pip or another Python package manager

#### Installation Steps

1. Install the package using your preferred package manager:
   ```bash
   pip install docgen
   ```
2. Run the first-time setup to configure your LLM provider, API keys, and accept the terms of service:
   ```bash
   docgen setup
   ```
